# Contributing

Thank you for helping maintainers spend less time on repetitive triage.

## Development

```bash
pnpm install
pnpm typecheck
pnpm test
pnpm build
```

## Pull requests

- Keep PRs small and focused.
- Explain the problem and the approach.
- Add or update tests for runtime behavior.
- Update docs when behavior changes.
- Do not add write-capable GitHub automation without a security discussion.
